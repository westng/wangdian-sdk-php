# 更新日志

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范。

## 贡献指南

如果您想为更新日志做出贡献，请：

1. 在提交代码时使用清晰的提交信息
2. 遵循 [约定式提交](https://www.conventionalcommits.org/) 规范
3. 在 PR 中描述您的更改

### 提交信息格式

```
<类型>(<范围>): <描述>

[可选的正文]

[可选的脚注]
```

### 类型说明

- `feat`: 新功能
- `fix`: 修复
- `docs`: 文档更新
- `style`: 代码格式调整
- `refactor`: 代码重构
- `test`: 测试相关
- `chore`: 构建过程或辅助工具的变动
